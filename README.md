Тема: Перевантаження функцій в C++

У роботі наведені приклади перевантаження функцій у мові C++.

1. Перевантаження за типом аргументів

Проєкт Overload_By_Type демонструє функції show(), що відрізняються типом переданого параметра.

2. Перевантаження за кількістю аргументів

Проєкт Overload_By_Count містить два варіанти функції sum(), що відрізняються кількістю параметрів.

Обидва приклади показують використання поліморфізму через механізм перевантаження функцій.
